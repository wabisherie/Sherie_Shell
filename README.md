# Sherie_Shell

   This is a UNIX like shell written in C.
  
  HOW IT WORKS
 
  To compile, use the following script
   gcc -Wall -Werror -Wextra -pedantic *.c -o Sash
 
  Afterwards you will see the new shell which I called Sash. (She's a girl by the way ;-) )
 
  To use it,
          ./Sash
